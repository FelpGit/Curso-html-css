Python

Exemplo de um código básico de Par ou Ímpar em linguegem Python:

        
        nu = int(input("Digte um número"))
        if num % 2 == 0
            print(f"O número {nu} é PAR!")
        else:
            print(f"O número {nu} é ÍMPAR!")
        print("Fim do programa")

Esse programa irá mostrar ao usuario se o número colocado é Par ou Ímpar.

A Simplificação desse código é mais impressionante que parece:
